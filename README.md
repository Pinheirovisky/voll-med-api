# voll-med-api
An API built with JAVA, to a Medical Bussiness Model.

## These endpoints have already been added:
### MEDICOS - /medicos
- **GET** - List all doctors;
- **POST** - Create one doctor;
- **PUT** - Update one doctor;
### PACIENTES - /pacientes
- **GET** - List all patients;
- **POST** - Create one patient;
